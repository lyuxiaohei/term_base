# 孤立术语

> 没有任何其他术语引用的术语（无入链）。这些术语可能需要补充关联关系。

```dataview
TABLE WITHOUT ID
  file.link AS "术语名称",
  system AS "所属系统",
  updated AS "更新时间"
FROM "term_vault"
WHERE !length(file.inlinks)
SORT system ASC, file.name ASC
```

## 统计

```dataview
TABLE WITHOUT ID
  length(rows) AS "孤立术语数量"
FROM "term_vault"
WHERE !length(file.inlinks)
```
