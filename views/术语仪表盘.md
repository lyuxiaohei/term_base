# 术语仪表盘

> 术语库整体健康状况一览。

## 总量统计

```dataview
TABLE WITHOUT ID
  length(rows) AS "术语数量"
FROM "terms"
GROUP BY system
SORT length(rows) DESC
```

## 最近更新（7天内）

```dataview
TABLE WITHOUT ID
  file.link AS "术语",
  updated AS "更新日期",
  system AS "系统"
FROM "terms"
WHERE date(updated) >= date(today) - dur(7 days)
SORT updated DESC
```

## 无英文名的术语

```dataview
TABLE WITHOUT ID
  file.link AS "术语",
  system AS "系统"
FROM "terms"
WHERE !en OR en = ""
SORT file.name ASC
```

## 关联最多的术语（Top 10）

```dataview
TABLE WITHOUT ID
  file.link AS "术语",
  length(file.outlinks) AS "出链数",
  length(file.inlinks) AS "入链数",
  system AS "系统"
FROM "terms"
SORT length(file.outlinks) DESC
LIMIT 10
```
