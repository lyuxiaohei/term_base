# 术语库

> 业务术语智能管理系统

## 快速入口

- [[术语总览]] — 查看全部术语和统计
- [[按系统查看]] — 按业务系统分类浏览
- [[最近更新]] — 最近修改的术语
- [[术语仪表盘]] — 健康状况与数据概览
- [[孤立术语]] — 没有被引用的术语
- [[log|操作日志]] — 自动化运行记录

## 操作指引

| 操作 | 方法 |
|------|------|
| 提取术语 | 将音频/文档放入 `src/pending/`，自动或手动运行 `python scripts/automate.py --now` |
| 新建术语 | 在 `terms/` 下新建笔记，参考 [[术语模板]] |
| 查看关系 | 点击术语中的 `[[双链]]` 或打开 Graph View |
| 健康检查 | 运行 `python scripts/lint.py` |
