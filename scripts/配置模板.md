填写完成后，将对应值填入 `scripts/config.json`。

## 1. AI 大模型 API

| 字段 | 说明 | 示例 |
|------|------|------|
| `base_url` | API 地址（OpenAI 兼容接口） | `https://api.deepseek.com/v1` |
| `api_key` | API 密钥 | `sk-xxxxxxxxxxxxxxxx` |
| `model` | 模型名称 | `deepseek-chat` |

**常见服务商配置：**

| 服务商 | base_url | model |

## 2. 邮箱（用于术语变更审批通知）

### 2.1 发件邮箱（公司邮箱或个人邮箱）

| 字段 | 说明 | 示例 |
|------|------|------|
| `smtp_host` | SMTP 服务器地址 | `smtp.qq.com` |
| `smtp_port` | SMTP 端口 | `465` |
| `sender` | 发件邮箱地址 | `yourname@qq.com` |
| `password` | 邮箱授权码（非登录密码） | `abcdefghijklmnop` |
| `imap_host` | IMAP 服务器地址（接收审批回复） | `imap.qq.com` |
| `imap_port` | IMAP 端口 | `993` |

**常见邮箱配置：**

| 邮箱 | smtp_host | imap_host | smtp_port | imap_port |
|------|-----------|-----------|-----------|-----------|
| QQ 邮箱 | `smtp.qq.com` | `imap.qq.com` | 465 | 993 |
| 163 邮箱 | `smtp.163.com` | `imap.163.com` | 465 | 993 |
| 腾讯企业邮 | `smtp.exmail.qq.com` | `imap.exmail.qq.com` | 465 | 993 |
| Gmail | `smtp.gmail.com` | `imap.gmail.com` | 465 | 993 |

> QQ邮箱授权码获取：设置 → 账户 → POP3/IMAP/SMTP → 生成授权码

> 填写你的：
> smtp_host = `________________`
> smtp_port = `________________`
> sender    = `________________`
> password  = `________________`
> imap_host = `________________`
> imap_port = `________________`
